ASSESSMENT REFLECTIVE JOURNAL:

MY PERFORMANCE:

Considering that I have gone through similar things when I started teaching myself web devlopment since 2013, coping with all the tasks in the weekly lab exercises wasn't a new thing to me. Because I was already familiar with the basics and or required elements that we were taught in class, it was easier for me to grasp everything and quickly put them into action and produce what was required of me.

However, even though that was the case, I still had a hard time trying to get used to other programs that I may or may not know how to use such as Adobe Dreamweaver. Even though I trained and familiarized myself with a number of different programs used in web development such as, Adode Dreamweaver, Sublime Text, Atom, Notepad++, Microsoft Expression Web 4, Adobe Edge Code & my personal favorite, Brackets, I still preferred text-based programs such as Brackets which do not provide us with a graphical user interface that lets us use a Design feature because the only freedom that such programs give us is writing each and every single line of code.

I believed and I still do, that practicing web development with a text-based program such as Brackets is much better than using a graphical user interface program like Adobe Dreamweaver. So, I'm sure you can guess that it was a challenge for me when I learned that some of the lab exercises were to be completed by using Adobe Dreamweaver since it has a built-in FTP feature that lets us automatically upload our webpages to the SOC (soc.uwl.ac.uk) server. But in the end, the practice that I had in my previous attempts with Adobe Dreamweaver meant that I was already used to the interface hence once again, I quickly adopted to what I was supposed to do and accomplised my tasks as required.
