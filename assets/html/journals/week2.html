WEEK 2 REFLECTIVE JOURNAL:

DEVELOPING A DIGITAL IDENTITY (DDI):

During week 2, we were introduced to Adobe Dreamweaver and some of its core features such as the FTP file transfer service. We had to connect to the university's server (SOC server) through the FTP service in Dreamweaver and then create new HTML and CSS documents which were then automatically uploaded to the server once we saved them. We also looked at using the W3.org validation service to check if our web pages were free of any errors that could compromise them. After having done that, we to created our second entries for the weekly reflective journal which we then ulpoaded to the server using Dreamweaver's FTP service once again.

INFORMATION SYSTEMS & DATABASES (ISDB):

We were introduced to Oracle's SQL for the first time where we were required to download the Oracle software and using our Student ID's, we had to connect to the SOC server where we linked to the itcentre database which contains sample tables and records which we used for practice during our tutorials in class and assignment practice back home. We were also provided with some information that we had to read through in order to better understand SQL as well as sample database questions for practice.

BUSINESS ACCOUNTING FOR MANAGERS (BAM):

We learned about the differences between Business and Financial Accounting, the advatanges or benefits of performing Financial Accounting. We also looked at the possible challenges that would a business is likely to face if it doesn't conduct any Financial Accounting throughout their business. We went further by looking at the accounting equation which is ASSETS = EQUITY + LIABILITIES which is used assess business situation when referring to the Statement of Financial Position.
