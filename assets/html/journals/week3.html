WEEK 3 REFLECTIVE JOURNAL:

DEVELOPING A DIGITAL IDENTITY (DDI):

In week 3 of DDI we learnt about the HTML Structure as well as the CSS syntax. We looked at the opening and closing tags of HTML which basically mean that any content withing the website needs to be enclosed in these tags for the web browser to understand what we need to display. However, there are also empty tags which do not require closing, examples of such tags are; img, br, hr & link etc. We also looked at the CSS syntax which includes a selector and a declaration which contains property that defines what value of the propert we want to affect and a value which refers to how we want that property should be in order to affect the selector on the web page.

INFORMATION SYSTEMS & DATABASES (ISDB):

During week 3 in the ISDB we learnt what ERD - Entity Relationship Diagrams are and what they are used for. ERD diagrams are used to show or illustrate a design of a database so that a Database developer can use them as references when developing the database. Usually, all entities in an ERD diagram will eventually become tables in the database whereas their attributes will become columns of that Table. After we went through the basics of what ERDs are, we did some practices by designing a few ER Diagrams using some sample question provided to us in class. This was to help us better understand what should be done and it also acted as a practice for the assignment since we are supposed to do something similar.

BUSINESS ACCOUNTING FOR MANAGERS (BAM):

We briefly looked at the different financial statements which are Statement of Financial Position, Income Statement & Cash Flows Statement. We then covered the statement of financial position in detail by trying to see what it is used for, how to prepare one, the issues and problems that a company is likely to face in case it doesn't prepare one or doesn't preapre it properly as well as the benefits of a properly prepared statement. We also looked at the double entry system and then we later did some practice with it as well as the statement of financial position.
